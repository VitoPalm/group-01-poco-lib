◆ handleUserLend()

void poco.company.group01pocolib.mvc.controller.UserTabController.handleUserLend ( )
private

Allows to lend a book to the selected user when the "Lend To" button is clicked on the User tab.

This method checks if a book was selected before selecting the user and pressing the button. If that is case, it moves to the Lending tab with the selected user and book pre-selected, only requiring the return date to be set.

If no book was selected, it moves to the Book tab to allow selection of a book to lend to the user.
