◆ handleBookLend()

void poco.company.group01pocolib.mvc.controller.BookTabController.handleBookLend ( )
private

Allows to lend the selected book when the "Lend" button is clicked on the Book tab.

This method checks if a user was selected before selecting the book and pressing the button. If that is case, it moves to the Lending tab with the selected user and book pre-selected, only requiring the return date to be set.

If no user was selected, it moves to the User tab to allow selection of a user to lend the book to.

◆ initialize()

void poco.company.group01pocolib.mvc.controller.BookTabController.initialize ( )
private

Initializes the controller class, setting up all the listeners. This method is automatically called after fxml file has been loaded.

Setup of all the listeners of the BookTab: selected table entry, Omnisearch textfield

  • When an entry is selected, the "Lend" and "View/Edit" buttons will become clickable
  • When the Omnisearch textfield is empty, the full table data is shown, whereas a type in the search box enables the view of the search results
  • When the Omnisearch textfield is empty, the prompt text shows the number of entries in the Set
  • When the window is resized to a tighter height, the pocologo is hidden
  • When the window is resized to a tighter width, the table resize policy becomes unconstrained to correctly visualize min. column sizes

Giovanni Orsini

Launches the View/Edit Book dialog.

This method opens a new dialog window for viewing or editing a book's details. It takes into account whether the book is new or existing, and whether editing is allowed.

Parameters
bookToEditThe book to view or edit. If null and isNewBook is true, a new book will be created.
isNewBooktrue if creating a new book, false if viewing/editing an existing book.
modeThe mode of the dialog, either VIEW, VIEW_ONLY, or EDIT.
