Detailed Description

Represents a Book in the library.

Each Book has a title, a list of authors, an ISBN, a release year, a number of copies owned, a number of copies currently lent out, a number for the amount of times the Book has been lent, and the line index for the DB file. The ISBN must be valid for the Book to be created. The book can be serialized for persistence.
