void poco.company.group01pocolib.mvc.controller.LendingPropController.setLending ( Lending lending)

Sets the lending to be edited/viewed in the dialog.

If the lending is null, a new empty lending is created to allow the creation of a new lending. When setting the lending, both the view labels and edit fields are updated with the lending details.

Parameters
lendingThe lending to set.
Exceptions
IllegalArgumentExceptionIf lending is null and no book or user is selected in the main controller.
