\doxysubsection{Detailed Description}
Represents a \doxylink{classpoco_1_1company_1_1group01pocolib_1_1mvc_1_1model_1_1Book}{Book} in the library. 

Each \doxylink{classpoco_1_1company_1_1group01pocolib_1_1mvc_1_1model_1_1Book}{Book} has a title, a list of authors, an ISBN, a release year, a number of copies owned, a number of copies currently lent out, a number for the amount of times the \doxylink{classpoco_1_1company_1_1group01pocolib_1_1mvc_1_1model_1_1Book}{Book} has been lent, and the line index for the \doxylink{classpoco_1_1company_1_1group01pocolib_1_1db_1_1DB}{DB} file. The ISBN must be valid for the \doxylink{classpoco_1_1company_1_1group01pocolib_1_1mvc_1_1model_1_1Book}{Book} to be created. The book can be serialized for persistence.
