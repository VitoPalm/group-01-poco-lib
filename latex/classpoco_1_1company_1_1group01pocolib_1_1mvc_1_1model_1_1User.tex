\doxysubsection{Detailed Description}
Represents a Library \doxylink{classpoco_1_1company_1_1group01pocolib_1_1mvc_1_1model_1_1User}{User}. 

Each \doxylink{classpoco_1_1company_1_1group01pocolib_1_1mvc_1_1model_1_1User}{User} has a unique ID, a name, a surname, an email, a number of currently borrowed books, and a total number of books borrowed ever. The ID and email must be of valid format for the object to be created. The class can be serialized for persistence.
