\doxysubsection{Detailed Description}
Allows creation and simple handling of a database (UTF-\/8 encoded file). 

This class is based on the assumption that the entries on the \doxylink{classpoco_1_1company_1_1group01pocolib_1_1db_1_1DB}{DB} are separated on different lines, but makes no further assumptions on how fields for a given entry are defined or separated. It allows reading, writing, and removing specific lines from a database file, it also includes a few methods to edit, remove and return lines based on regex patterns. ~\newline
~\newline
 The class preloads the \doxylink{classpoco_1_1company_1_1group01pocolib_1_1db_1_1DB}{DB} lines into an internal cache for faster access. The original file is always modified when writing or removing lines, and the cache is rebuilt accordingly. The class is built with the idea of never having an edit on the \doxylink{classpoco_1_1company_1_1group01pocolib_1_1db_1_1DB}{DB} file without updating the cache, so the two are always in sync. ~\newline
~\newline
 The reason for an additional data support for the application (which already obtains data permanence between sessions through serialization of objects) is reliability and, most importantly, the ability to support future changes to the base data models. The serialized objects are not human-\/readable, and any change to the data models would make it not possible to retrieve previously saved data. ~\newline
~\newline
 The \doxylink{classpoco_1_1company_1_1group01pocolib_1_1db_1_1DB}{DB} class, being reliant on a simple text file, allows easy "{}legacy support"{} by any future versions of the application, as well as easy manual editing of the data if needed. (and, if needed, importing and exporting data from and to other machines through these simple text files).
