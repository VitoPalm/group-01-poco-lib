\doxysubsection{Detailed Description}
Represents a trigram index for fast searching of items based on a provided {\ttfamily String}. 

This class allows adding, removing, and generating trigrams for searchable content associated with items of type {\ttfamily T}. The trigrams are stored in a map where each trigram maps to a list of items related to that trigram.

Generates trigrams from the provided searchable content. 

A trigram is a contiguous sequence of three characters from a given string. This method extracts all possible trigrams from the input string and returns them as a list. ~\newline
~\newline
 If the input string is shorter than three characters, it will generate all possible combinations of the Ngram available, that result into a trigram when adding bell characters as padding. ~\newline
~\newline
 The method does some basic preprocessing, such as converting the string to lowercase, and trimming whitespace.
